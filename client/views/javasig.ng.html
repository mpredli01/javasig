Building Web Apps with Meteor and AngularJS

Presented by Mike Redlich

The folks at AngularJS and Meteor have teamed up to create a library for developers to create stunning web apps utilizing both technologies. Advantages include:

  • Model-View-Controller (MVC) and Model-View-ViewModel (MVVM) structure on the client
  • Best real time backend for AngularJS
  • Super easy, real-time integration with the database
  • Full stack and open source
  • Easy deployment

This presentation will demonstrate how to build a new web app by starting with some basics and then adding functionality along the way.

Technical Overview of Amazon Echo

Presented by Barry Burd

Echo is Amazon's entry in the intelligent voice-enabled assistant market. Unlike Siri or Google Now, Echo isn't mobile. Echo stays plugged into an outlet in your home. But Echo is completely hands-free, so you can talk to Echo the way Captain Picard talks to his ship's computer. In this presentation, Barry will demonstrate the Echo's capabilities and show you how to develop new capabilities for the Echo device. For a preview, click here.
