Tuesday, February 9, 2016

Why Rust? Why Not!

Presented by Paulmichael Blasucci

Modern high-level languages offer lots of great features. However, when it comes time to do systems-level programming, one is fast with some hard choices. Give up the higher-level features we all love (by using C), or get stuck in the morass that is C++.

But, wait!

Rust offers a third choice. It promises high-level expressivity with the power and control needed for systems programming. This talk provides a general introduction to Rust. It'll introduce the basic syntax, developer tools, and overall "feel" of the language. We'll also spend some time focusing on where and when it makes the most sense to use Rust.

Tuesday, January 12, 2016

Firebase 101

Presented by Bill Brutzman

Firebase is a cool new cloud NoSQL database from Google. In FireBase, data is stored as JSON synched to all connected clients in real time, that is, mobile app, tablet, and desktop users do not have to hit the browser refresh button. FireBase Includes built-in user authentication to e-mail, Facebook, Twitter, GitHub, and Google with built-in SSL and hosting. Developers can code via JavaScript, Java, REST, Objective C and can get started with FireBase for free. FireBase code will be demonstrated with an example website using Data Structures, CRUD, UserAccess, Security, Latency, Presence, and Deployment.

Processing JSON with Java

Presented by Mike Redlich

JavaScript Object Notation, i.e., JSON, is an open standard format that uses human-readable text to transmit data objects consisting of key–value pairs. Here is an example: 

                
                    {
                    "lastName": "Redlich",
                    "firstName: "Michael",
                    "address":
                        {
                        "streetAddress": "123 Anywhere Street",
                        "city": "Flemington"
                        },
                    "state": "New Jersey",
                    "phoneNumber":
                        [{
                        "type": "home",
                        "number": "(908) 000-0000"
                        },
                        {
                        "type": "cell",
                        "number": "(908) 000-0000"
                        }]
                    }

JSON is used in a wide variety of applications so processing is, of course, a necessity. We will review three applications for processing JSON:

  • json-simple
  • Jackson
  • gson

November 2015 Meeting

Tuesday, November 10, 2015

Building Web Apps with Meteor and AngularJS

Presented by Mike Redlich

The folks at AngularJS and Meteor have teamed up to create a library for developers to create stunning web apps utilizing both technologies. Advantages include:

  • Model-View-Controller (MVC) and Model-View-ViewModel (MVVM) structure on the client
  • Best real time backend for AngularJS
  • Super easy, real-time integration with the database
  • Full stack and open source
  • Easy deployment

This presentation will demonstrate how to build a new web app by starting with some basics and then adding functionality along the way.

Technical Overview of Amazon Echo

Presented by Barry Burd

Echo is Amazon's entry in the intelligent voice-enabled assistant market. Unlike Siri or Google Now, Echo isn't mobile. Echo stays plugged into an outlet in your home. But Echo is completely hands-free, so you can talk to Echo the way Captain Picard talks to his ship's computer.
