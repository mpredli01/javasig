Source Code Archive

The source code archive is a repository of zip files containing source code and the necessary build files (build.xml, build.gradle, SQL scripts, etc.), for most of the monthly meetings. Each zip file corresponds to a monthly meeting, and is named with the format, jsigγγμμ.zip, where γγ represents the two-digit year and μμ represents the two-digit month. Each monthly meeting has its own IntelliJ IDEA project and has been built with Ant and/or Gradle , both Java-based build tools for compiling and running Java applications (and a whole lot more).
